2019 International Science fairs

In this project, we consider the following discrete dynamic system: Given n arbitrary points
on the circle dividing the circle into n arcs. In one transition, each point jumps simultaneously from its current state counterclockwise to the ratio : p q division point of the arc in front of it. Repeat the process indefinitely. If there is a point that can return to its initial position after m transitions, we will call m as the returning time.
Goal: (i) Find all the returning time m. (ii) Analyze the long-term behavior of the system.

Conclusion:
1. We discuss the smallest returning time * m and all large enough m, when there is a point returning to its initial position after m transitions.
2. Given n, p, q and large enough t, we can determine whether or not there exists an initial state for the dynamic system to return with the number of transition t.
3. We have shown that a dynamic system of our interest always includes a sub-process whose limiting steady state exists and they divide the circle evenly.
